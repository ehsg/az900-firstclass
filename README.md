Public Cloud vs Private Cloud vs Hybrid Cloud
-Public Cloud = Third-party managed resources over internet, offering infrastructure for better scability and cost prediction
-Private Cloud = On-premises servers hosted inside the company requiring bigger initial investment, offering more control and security
-Hybrid Cloud = Combine both worlds, offers the flexibility to keep some services on-premises and scalate others on the public cloud as Azure

Opex vs Capex
Opex = Expense based on daily usage or required to maintain operations
Capex = Investment to increase capability, renew/upgrade resources
